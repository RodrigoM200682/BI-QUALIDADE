# BI Qualidade Integrado — versão corrigida

Correção aplicada para FileNotFoundError no Streamlit Cloud.

## Como publicar
1. Extraia o ZIP.
2. Envie todo o conteúdo para o repositório GitHub.
3. O arquivo principal deve ser `app.py`.
4. A pasta `modulos` deve ir junto. Mesmo se ela faltar, o app agora recria os módulos automaticamente.

## Observação
As funcionalidades dos três aplicativos originais foram preservadas em módulos independentes.
